## About the Project

My grandmother is 91 years old. She was born in Corrientes, Argentina, worked as a teacher her whole life, traveled the world, danced, and made artisan crafts. She recently started writing a book â€” not a memoir, but a collection of essays with her opinions and wisdom on topics that matter to her: education, politics, the economy, carnavales, travel, and more.

I built **Hablemos en 10** as a birthday gift for her. It's a tablet-friendly web app that serves as a guide for her handwriting sessions. She doesn't type â€” she looks at the screen, gets inspired by a topic, and writes by hand in her physical book.

### How It Works

1. She opens the app and sees a warm, personalized greeting
2. She taps **"Dame un tema"** (Give me a topic)
3. The app generates a topic powered by AI, with keywords and guiding questions â€” deeply grounded in Corrientes culture, Argentine history, and her personal interests
4. She taps **"Empezar a escribir"** and a 10-minute timer starts
5. She writes in her book while the timer counts down
6. When time's up, the topic is saved to her history

The AI-generated topics are contextually aware: they reference local traditions like the Carnavales de Corrientes, Argentine economic history, education from her era, world figures she follows on TV, and the places she's traveled.

## Design Principles

This app was designed for a specific user: a 91-year-old woman using a tablet. Every design decision reflects that:

- **Minimum 18px body text**, 24px+ for headings and timer
- **Maximum 2â€“3 actions** visible at any time
- **No scrolling required** on main screens
- **High contrast** warm palette (cream background, golden accents)
- **Typography**: Merriweather (headings) + Nunito (body) â€” warm, readable, literary
- **All UI in Argentine Spanish** with a familiar, loving tone

## Local Development

### Prerequisites

- Node.js 18+
- Yarn
- Firebase project with Auth + Firestore enabled
- Anthropic API key

### Setup

```bash
# Clone the repo
git clone https://github.com/andrealopezpalomeque/hablemos-en-10.git
cd hablemos-en-10

# Install client dependencies
cd client
yarn install

# Install server dependencies
cd ../server
yarn install
```

### Environment Variables

**Client** (`/client/.env`):
```env
NUXT_PUBLIC_FIREBASE_API_KEY=your_key
NUXT_PUBLIC_FIREBASE_AUTH_DOMAIN=hablemos-en-10.firebaseapp.com
NUXT_PUBLIC_FIREBASE_PROJECT_ID=hablemos-en-10
NUXT_PUBLIC_FIREBASE_STORAGE_BUCKET=hablemos-en-10.appspot.com
NUXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NUXT_PUBLIC_FIREBASE_APP_ID=your_app_id
NUXT_PUBLIC_API_BASE_URL=http://localhost:3001
```

**Server** (`/server/.env`):
```env
ANTHROPIC_API_KEY=your_key
FIREBASE_SERVICE_ACCOUNT=your_service_account_json
PORT=3001
ALLOWED_ORIGINS=http://localhost:3000
```

### Run
